The `@bluxcc/core` package is framework-agnostic â€” use it in any JavaScript or TypeScript project without a framework dependency.

## Installation

Install the Blux core SDK using your package manager of choice:

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab value="npm">
```bash
    npm install @bluxcc/core
```
  </Tab>
  <Tab value="pnpm">
```bash
    pnpm add @bluxcc/core
```
  </Tab>
  <Tab value="yarn">
```bash
    yarn add @bluxcc/core
```
  </Tab>
</Tabs>

## Setup

Call `createConfig` once at the entry point of your application to initialize Blux. Do not call it more than once.

<Tabs items={['ESM', 'CJS', 'HTML (CDN)']}>
  <Tab value="ESM">
```ts
    import { createConfig, core } from "@bluxcc/core";

    createConfig({
      appName: "My App",
      networks: [core.networks.mainnet, core.networks.testnet],
    });
```
  </Tab>
  <Tab value="CJS">
```ts
    const { createConfig, core } = require("@bluxcc/core");

    createConfig({
      appName: "My App",
      networks: [core.networks.mainnet, core.networks.testnet],
    });
```
  </Tab>
  <Tab value="HTML (CDN)">
```html
    <script src="https://cdn.jsdelivr.net/npm/@bluxcc/core/dist/index.iife.js"></script>

    <script>
      Blux.createConfig({
        appName: "My App",
        networks: [Blux.core.networks.mainnet, Blux.core.networks.testnet],
      });
    </script>
```
  </Tab>
</Tabs>
